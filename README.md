## How to run
1- Clone this repository <br>
2- Clone the back-end repository: https://github.com/lbritors/MyWalletBack <br>
3- Install dependencies
```
npm install
```

4- Run the front-end:
```
npm start
```
5- Access localhot:3000 on your browser
